# Contributing

Thank you for your interest in contributing! This repository is a community-curated collection of practical Claude skills that automate repetitive tasks, improve productivity, and streamline everyday workflows.

Whether you're contributing an original skill, improving an existing one, or recommending a useful public resource, your contributions are welcome.

## What You Can Contribute

We welcome contributions such as:

- New Claude skills
- Improvements to existing skills
- Better documentation
- Bug fixes and corrections
- New categories or organizational improvements
- Curated links to publicly available Claude skills

## Skill Guidelines

Each skill should include the following information:

- **Name**
- **Description** – What does the skill do?
- **Use Cases** – When should someone use it?
- **The Skill** – The prompt, configuration, or instructions.
- **Example** *(optional but encouraged)*

Keep skills focused, reusable, and well-documented.

## Repository Structure

```text
skills/
├── productivity/
├── coding/
├── research/
├── writing/
├── automation/
└── reasoning/
```

Please place your contribution in the most appropriate category.

## Attribution

If your contribution is based on someone else's work:

- Always credit the original author.
- Include a link to the original repository, article, or source.
- Only copy content if the original license explicitly permits redistribution.
- If your work is inspired by another skill but written from scratch, mention the inspiration.

This repository aims to celebrate and promote community contributions while respecting authorship and licensing.

## Pull Requests

Before opening a Pull Request, please ensure that:

- Your skill is clearly documented.
- The formatting matches the rest of the repository.
- The skill has been tested.
- Proper attribution has been provided where applicable.
- The contribution does not duplicate an existing skill.

## Reporting Issues

Found a mistake or have a suggestion?

Open an Issue describing the problem or proposed improvement. Feature requests and discussions are always welcome.

## Code of Conduct

Please be respectful, constructive, and welcoming. This project exists to make high-quality Claude skills easier to discover and share with the community.

Happy contributing!
